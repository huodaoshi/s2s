# 分拣标签

技能使用五个规范分拣角色表述。本文件将这些角色映射到本仓库 issue 跟踪器中的实际标签字符串。

| mattpocock/skills 中的角色 | 本跟踪器中的标签 | 含义 |
| -------------------------- | ---------------- | ---- |
| `needs-triage`             | `needs-triage`   | 维护者需评估该 issue |
| `needs-info`               | `needs-info`     | 等待报告者补充信息 |
| `ready-for-agent`          | `ready-for-agent`| 规格完整，可供 AFK Agent 领取 |
| `ready-for-human`          | `ready-for-human`| 需人工实现 |
| `wontfix`                  | `wontfix`        | 不予处理 |

技能提到某角色时（如「应用可供 AFK 领取的分拣标签」），使用本表右侧对应的标签字符串。

请编辑右列以匹配你实际使用的词汇。
